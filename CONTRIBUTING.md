# vgstation14 Contributing Guidelines

Thanks for contributing to vgstation14.

vgstation14 is a fork of Space Station 14. The codebase and engine are shared
with upstream, so the upstream [codebase conventions](https://docs.spacestation14.com/en/general-development/codebase-info/codebase-organization.html)
and [PR guidelines](https://docs.spacestation14.com/en/general-development/codebase-info/pull-request-guidelines.html)
still apply.

**On top of those, vgstation14 has one rule upstream does not:** all divergence
is isolated. Before opening a PR, read [AGENTS.md](AGENTS.md) and
[docs/vg/divergence-policy.md](docs/vg/divergence-policy.md) — additive work goes
in `_VG/` folders, and edits to upstream files must be recorded in
[docs/vg/divergence-registry.md](docs/vg/divergence-registry.md).
