GroupId

orGroup signifies to pick between entities designated with an ID.

To define an orGroup in a StorageFill component you need to add it to the entities you want to choose between and add a prob field. In this example there is a 50% chance the storage spawns with Y or Z. 

- type: StorageFill
contents:
- name: X
- name: Y
prob: 0.50
orGroup: YOrZ
- name: Z
orGroup: YOrZ

MaxAmount

How many of this can be spawned, in total. If this is lesser or equal to Amount, it will spawn Amount exactly. Otherwise, it chooses a random value between Amount and MaxAmount on spawn.
