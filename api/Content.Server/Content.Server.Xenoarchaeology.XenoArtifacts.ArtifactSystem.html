Remarks

General balancing (for fully unlocked artifacts): Simple (1-2 Nodes): ~10K Medium (5-8 Nodes): ~30-40K Complex (7-12 Nodes): ~60-80K

Simple artifacts should be enough to unlock a few techs. Medium should get you partway through a tree. Complex should get you through a full tree and then some.
